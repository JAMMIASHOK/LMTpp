/*!
\generic_comment get_interp

    Cette fonction permet de calculer l'image d'un point par une fonction définie comme combinaison linéaire des fonctions de forme de l'élément de référence (Bar, Triangle, Triangle_6, Quad, Tetra...).
    Description des paramètres de la fonction get_interp() :
    * le premier paramètre est le type d'élément.
    * le deuxième est le type d'interpolation. Il y a six classes standard, Global, Nodal, Elementary, Gauss, Der_nodal et Skin_elementary. Les trois premières sont les plus importantes.
        * \a Global est utilisée lorsque la fonction est constante sur tout le maillage. Dans ce cas, le tableau val ne contient qu'une seule valeur.
        * \a Nodal est utilisée quand la fonction est définie par ses valeurs sur les neuds de l'élément. val doit contenir autant de valeurs qu'il y a de noeuds à lélément.
        * \a Elementary sert lorsque la fonction est constante sur l'élément. val ne contient donc qu'une seule valeur.
    * le troisième paramètre var_inter contient les coordonnées "barycentrique" dans l'espace de référence du point dont on veut calculer l'interpolation.
    * le quatrième paramètre, val, est le tableau des valeurs de la fonctions qui sert à l'évaluation (interpolation). val peut donc être de la forme  Vec<A> où A est une classe représentant un élément d'un espace vectoriel réel : i.e. on doit avoir les opérations a1+a2, a1-a2, l*a1 où l est un réel et a1, a2 des élément de A.
    * le dernier paramètre est le résultat de l'appel de get_interp.

    \relates Vec
    \keyword Maillage/Elément/Opération
    \friend hugo.leclerc@lmt.ens-cachan.fr
    \friend raphael.pasquier@lmt.ens-cachan.fr

*/
